Hexo博客搭建攻略
  1. 前置条件
  2. 第一步: 安装hexo框架并生成博客
  3. 第二步：配置hexo主题

前置条件

  • 已配置好node环境
  • 已拥有github账号,并熟悉操作

第一步: 安装hexo框架并生成博客

  • 使用npm安装hexo框架： npm install hexo-cli -g
  • 到文件目录下使用hexo初始化博客：hexo init 你的博客文件名
  • 进入生成好的博客：cd 你的博客文件名
  • 安装博客依赖：npm install
  • 确认博客是否可以运行：hexo server (博客默认运行在本地4000端口)
  • 恭喜到这一步，博客已经运行起来了(如需深入了解,请阅读 https://hexo.io/zh-cn/index.html)

第二步：配置hexo主题

  • 在博客目录下安装主题：npm install hexo-theme-keep (本文使用的主题为keep，也可选择其他主题，详细请阅读上一步网站)
  • 从博客根目录找到_config.yml(该文件是hexo的配置项),进入文件修改hexo主题为keep,将theme的值修改为keep即可
  • 重启博客确认主题是否应用成功
  • 恭喜到这一步,主题已经安装成功(如需深入了解,请阅读: https://keep-docs.xpoet.cn/ )
